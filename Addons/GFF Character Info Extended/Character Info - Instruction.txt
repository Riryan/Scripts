									  Installation and Configuration

1. Disable or Delete the "CharacterInfo" from Canvas

2. Drag the "GFF CharacterInfo.prefab" from (Addons\GFF Character Info Extended\Prefabs\) to the canvas element so canvas is the parent.

3. Select the "Shortcuts" object in Canvas. In the inspector Drag in "Character Info Panel" the object "CharacterInfoPanel" in Canvas\GFF CharacterInfo Extended\CharacterInfoPanel
