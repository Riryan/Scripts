                              Installation and Configuration

1. Edit the "Energy.cs"  Script in ō/Scripts/" 
   a)    find line      public abstract class Energy : NetworkBehaviour
         change to      public abstract partial class Energy : NetworkBehaviour


2. Edit the "EquipmentItem.cs"  Script in ōAssets/uMMORPG/Scripts/ScriptableItems/" 
   a)     Find line       public class EquipmentItem : UsableItem
          change to       public partial class EquipmentItem : UsableItem

   b)   Before line       return tip.ToString();
           add line       Utils.InvokeMany(typeof(EquipmentItem), this, "ToolTip_", tip);


3. Edit the "Combat.cs"  Script in ōAssets/uMMORPG/Scripts/" 
   a)   before line       victim.health.current -= damageDealt;
           add line       if (entity is Player player) player.combatSkills.HitEnemy(victim, damageDealt);

   b)   before line       onDamageDealtTo.Invoke(victim);
           add line       if (victim is Player targetPlayer) targetPlayer.combatSkills.ReceivedDamage(damageType);


   c)    find line      return baseDamage.Get(level.current) + bonus;
         change to      if (entity is Player player) return baseDamage.Get(level.current) + bonus + player.combatSkills.GetDamageBonus(baseDamage.Get(level.current) + bonus);
                        else return baseDamage.Get(level.current) + bonus;


   d)    find line      return baseDefense.Get(level.current) + bonus;
         change to      if (entity is Player player) return baseDamage.Get(level.current) + bonus + player.combatSkills.GetDefenseBonus();
                        else return baseDefense.Get(level.current) + bonus;


4. Edit the "WeaponItem.cs"  Script in ō/Scripts/" 
   a)    find line      public class WeaponItem : EquipmentItem
         change to      public partial class WeaponItem : EquipmentItem


5. Edit all player prefabs (uMMORPG\Prefabs\Entities\Players\)
   a) add scripts "PlayerCombatSkills"
   b) in Inspector in script "PlayerCombatSkills" in "Skill Templates" add Combat skiils from (Addons\GFF Character Characteristics - Combat Skills\Resources\)


6. Edit the "Health.cs"  Script in ō/Scripts/" 
   a)    find line      return baseThisLevel + bonus;
         change to      if (entity != null && entity is Player player) return baseThisLevel + bonus + player.combatSkills.BonusHealth();
                        else return baseThisLevel + bonus;


7. Edit the "Mana.cs"  Script in ō/Scripts/" 
   a)    find line      return baseThisLevel + bonus;
         change to      if (entity != null && entity is Player player) return baseThisLevel + bonus + player.combatSkills.BonusMana();
                        else return baseThisLevel + bonus;


8. All items in folder "Assets/uMMORPG/Resources/Equipment/" have characteristics added
  * you need to fill them for weapons "Weapon type"


9. Configure addon GFF Character Info Extended




Melee - add Sp and Damage
Range - add Sp and Damage
Force - add Fp and Damage
Defense - add Hp and Defense
Shield - add Block Chance
